## pastinn (Pascal Tiny Neural Network)

This is a Pascal port of [tinn](https://github.com/glouw/tinn) (the tiny neural network)

## Features

 - Portable - can be compiled with Delphi or FreePascal
 - Sigmoidal activation
 - One hidden layer
